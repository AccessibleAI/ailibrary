Options the library supports:

1. Resizing
    --height   V
    --width    V
    --channels V

2. add noise
    --add_noise
        'gaussian' (mean-0, variance-1)
    
3. Remove noise (De-noising)
    --denoise

4. Segmentation (smoothing edges)
    --segmentation

5. blurring
    --blur

6. zip (true/false)
    --zip_all
