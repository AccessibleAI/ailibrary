Options the library supports:

1. Resizing
    --height   V

2. add noise
    --add_noise
    
3. Remove noise (De-noising)
    --denoise

5. blurring
    --blur

6. zip (true/false)
    --zip_all
    
6.5 convolution
    --convolve (list of lists)
